
# Flight Price Prediction App ✈️

This is a **Flight Price Prediction App** built using **Random Forest** and **Streamlit**.  
Users can select flight details like departure/arrival time, airline, source, destination, and total stops to predict flight ticket prices in India.

## Features
- Predict flight prices using a trained Random Forest model
- Handles multiple airlines, sources, and destinations
- Calculates duration automatically
- Clean and interactive **Streamlit front-end**
